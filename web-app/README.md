# Nervous ECG web application

## Description

This folder contains a html/javascript application to connect to the ECG sensors.
THe application plot ECG data in real-time and save signal into .csv file.*

## Usage

Open ```index.html``` in a chromium-based web browser (tested with Chrome and Microsoft Edge)
When using the build executable instead of the static web page (see below), the application will connect to the first sensor scanned instead of displaying a list of available sensors.

## Build

This application can be build into Windows / Linux executable using Electron.
